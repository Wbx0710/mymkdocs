# 半导体物理

!!! danger "半导体物理"
    * 学分：3
    * 任课教师：程然

!!! info "课程基本信息"
    * 成绩构成：
    	* 考勤：10%
    	* 作业：40%（会看正确率，8个作业，越到后面越难；但是和PPT上或书上例题相关度大）
    	* 期中考试：20%
    	* 期末考试：30%
	* 全英文：授课、课件、作业以及考试全部英文
	* 期中考试：这阶段学习的内容主要为晶体的一些知识、一些基础的量子力学和基础的能带理论
	* 期末考试：后半部分学习则是利用能带知识分析具体器件（二极管、三极管、mos管等等），基本不会再考察期中考试的内容（能带除外，毕竟这个太基础了）
